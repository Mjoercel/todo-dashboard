# TODO Dashboard - Vercel Deployment

## SCHNELLE ANLEITUNG (5 Minuten)

### Option 1: Direkt auf Vercel deployen (EINFACHST)

1. Gehe zu: https://vercel.com/new
2. Logge dich mit GitHub ein
3. Ziehe diesen kompletten Ordner in das Upload-Feld
4. Klicke "Deploy"
5. Warte 30-60 Sekunden
6. FERTIG! Dein Dashboard ist live!

### Option 2: Via GitHub (für Updates später)

1. Erstelle neues Repository auf GitHub
2. Lade alle Dateien hoch
3. Gehe zu Vercel → "Import Project"
4. Wähle dein GitHub Repository
5. Klicke "Deploy"

## Deine Dashboard-URL

Nach dem Deployment bekommst du eine URL wie:
`https://todo-dashboard-xyz123.vercel.app`

Diese URL kannst du:
- Bookmarken
- Von überall aufrufen
- Mit niemandem teilen (bleibt privat)
